Fleet Vehicle Request by Employee

A request of Fleet / Vehicle by Employee Odoo App

This app allows your employee(s) to request for Fleet or Vehicle in the company for a specified period. Requests will go to the fleet team for approval and the Fleet User / Fleet Manager can approve requests for fleet and assign fleet to employees.


Main Features:

  • This app allows your employees to create fleet vehicle requests as shown.
  • PDF report of employee fleet requests is also available as shown in the sample below.
  • Workflow of approval by Fleet User/Fleet Administrator and assignment and return of the fleet.
  • For more details please watch the below screenshots and watch the video.

Workflow of Module: New -> Confirmed -> Approved -> Assigned -> Returned


Flow of Fleet Vehicle Request:

  • Employees can create fleet vehicle requests.
  • Employees can confirm fleet vehicle requests.
  • Fleet User/Administrator can approve fleet vehicle request of employee.
  • Fleet User/Administrator can assign fleet vehicle requests of employee.
  • Fleet User/Administrator can return fleet vehicle request of an employee.
  • Print PDF report of employee fleet requests by users.
  • Note: Vehicle request for (field) filled up by Fleet Administrator before approval.
